# Clasna Travel PRO

Enhanced, visually rich travel site demo inspired by modern travel agencies (like Azerways).
- Node.js (Express) backend with JSON data store
- Frontend: Bootstrap 5, Swiper.js hero slider, AOS scroll animations, FontAwesome icons
- Admin panel to add tours (header x-admin-pass required)
- Sample tours included

Quick start:
1. unzip and `cd` into folder
2. `npm install`
3. `ADMIN_PASS=YourSecret npm start`
4. Open http://localhost:3000
